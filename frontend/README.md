## Reservation app front

### This app include the following features:

- react.js
- sass
- moment
- reactstrap
- axios
- big-calendar


## Commands

### Run

```zsh
# Install modules / Run 
$ yarn
$ yarn start
```

### Test

```zsh
# Test
$ yarn test                           # Run all test
```

### Build

```zsh
# Build
$ yarn build
```
